oxi

A modern, fast, and secure open-source webmail client. Feature-complete Roundcube replacement that connects to any IMAP/SMTP server.

Next.js Rust IMAP/SMTP Tantivy Search SQLite Docker

Fast

Rust backend with async I/O. SQLite cache and Tantivy full-text search for instant results across thousands of emails.

Secure

Memory-safe Rust. No PHP attack surface. HTML sanitization, CSP headers, PGP/S-MIME encryption support.

Modern UI

Clean React frontend with Next.js and Tailwind. Thread view, drag-and-drop, keyboard shortcuts, mobile-responsive.

Drop-in

Connects to any existing mail server. Single Docker image. No bundled mail server, no vendor lock-in.
